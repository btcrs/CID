# Carothers Interior Design Website

A modern, minimalist one-page website built with Next.js, TypeScript, and Vanilla CSS.

## Features
- **Responsive Design**: Mobile-friendly layout.
- **Sectioned Navigation**: Easy access to Services, Portfolio, About, and Contact.
- **High-End Aesthetic**: Minimalist color palette and architectural typography.
- **Vercel Ready**: Optimized for seamless deployment on Vercel.

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`

## Deployment
This project is optimized for **Vercel**. 
To deploy:
1. Push this project to a GitHub repository.
2. Connect the repository to your Vercel account.
3. Vercel will automatically detect the Next.js project and deploy it.
